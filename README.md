# git_study
깃 공부한것입니다.
- git 명령어 모음
    
    git init
    
    git add (.은 전체 아니면 파일이름)
    
    git commit -m “커밋명” (commit할때의 이름을 지정해주는거)(중간저장 느낌)
    
    git remote add origin 원격주소
    
    git branch (ex) sub 브랜치명)
    
    git branch
    
    git push origin (sub←- 브랜치명 (넣어주는거 ))
    
    git pull origin (sub ←-브랜치명(가져오는거))
